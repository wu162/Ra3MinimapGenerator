## 风格
每种风格都应相应的陆地层数限制
   * 草地    4层
   * 雪地    3层
   * 要塞    2层

## 如何获得好的结果
* 陆地高度一定要**高于**200，水面高度要为200，有海岸的陆地高度最好为210（至少不要高于230）
* 地图的陆地边缘一定要刷不可通过区域，且厚度最好大于3，该生成器是基于此信息来生成边缘
